Introduction To the Simulator

Toolbar

The toolbar consists of 5 buttons:

  1. Cursor- It will let us select and drag any elements and make connections betweens nodes
  2. + Button- On selecting it adds a new node where the the cursor is by clicking mouse 1
  3. Delete Button- On selecting it deletes the node where the the cursor is by clicking mouse 1
  4. Grid - Toggles the grid in the main canvas
  5. Theme - Toggles between night and dark mode
